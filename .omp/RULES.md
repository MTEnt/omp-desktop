# OMP Desktop project rules

## Impeccable UI standard

All UI work in this repo MUST follow Impeccable (https://impeccable.style/docs/).

1. Read `skill://impeccable` before designing or editing UI.
2. Run `node .agents/skills/impeccable/scripts/context.mjs` once per session from the project root.
3. For polish/critique/audit/craft/shape/layout/typeset/animate intents, load the matching `reference/<command>.md`.
4. Obey absolute bans (no AI-slop defaults). Match the existing titanium dark cockpit language rather than inventing a second aesthetic.
5. Prefer creating/updating PRODUCT.md + DESIGN.md via impeccable init/document when design context is missing.
